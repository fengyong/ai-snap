# AISnap

macOS 原生截图标注工具，基于 Swift + AppKit 构建，支持丰富的标注功能和直观的交互体验。

## 功能列表

### 截图捕获
- **区域截图** - 全屏覆盖层，鼠标拖拽选择任意矩形区域
- **窗口截图** - 自动检测鼠标下方的窗口并捕获
- **Retina 适配** - 自动识别屏幕 DPI，正确处理 2x/3x 分辨率

### 标注工具
- **箭头** - 拖拽绘制箭头标注，支持 6 种预设样式（实心、开放、虚线、菱形、圆端、点菱）
- **矩形** - 拖拽绘制矩形边框
- **圆形** - 拖拽绘制正圆（半径约束相等）
- **椭圆** - 拖拽绘制椭圆（宽高独立）
- **聚光灯** - 框选高亮区域，其余区域自动变暗（半透明遮罩），支持多个聚光灯叠加
- **表情贴纸** - 18 个预设表情/符号，单击放置到画布上

### 对象编辑
- **选择与移动** - 双图层 Color Picking 方案，点击精确选中对象，拖拽移动
- **取消选中** - 按 ESC 取消当前选中，恢复绘制模式
- **旋转** - Option + 拖拽旋转选中对象
- **缩放** - Shift + 拖拽缩放选中对象
- **删除** - Delete 键或点击右上角红色 X 按钮删除
- **撤销/重做** - Cmd+Z 撤销，Cmd+Shift+Z 重做，支持所有操作（添加、删除、移动、旋转、缩放）
- **选中特效** - 蓝色发光虚线边框 + 四角手柄 + 右上角删除按钮

### 智能辅助
- **点吸附** - 绘制时自动吸附到附近对象的端点、中心、角点、边中点（12px 阈值）
- **被动吸附指示器** - 鼠标悬停时实时显示最近吸附点，十字 + 菱形可视化提示
- **中心点绘制** - 起点吸附到已有对象时，以吸附点为中心向外绘制圆形/椭圆/矩形
- **对象附着** - 箭头端点可附着到形状周长，父对象移动/旋转/缩放时箭头自动跟随
- **级联删除** - 删除父形状时自动删除附着的箭头

### 颜色与样式
- **5 套调色板** - 鲜明、专业、柔和、高对比、灰度，一键切换
- **线宽** - 默认 15px 粗线条，清晰醒目
- **箭头样式** - 6 种预设箭头头部/尾部/线型组合

### 水印
- **文本水印** - 可配置水印文本内容
- **平铺模式** - 自动以 -30 度角平铺覆盖全图
- **导出时渲染** - 仅在保存/复制时叠加，不影响编辑

### 导出
- **保存为 PNG** - 通过系统保存面板选择路径
- **复制到剪贴板** - 一键复制，可直接粘贴到其他应用

### 界面
- **macOS 菜单栏** - 编辑菜单（撤销/重做/删除）、帮助菜单
- **直观工具栏** - 中文文字按钮 + 分组标签 + 悬浮提示
- **帮助对话框** - 详细的功能使用说明
- **窗口自适应** - 截图超出屏幕时自动缩放适配
- **Debug 面板** - 右侧实时显示 Layer B (Hit Test) 可视化

## 技术架构

| 组件 | 说明 |
|------|------|
| `main.swift` | 应用入口 |
| `AppDelegate.swift` | 状态栏菜单、截图调度 |
| `ScreenCapture.swift` | CGWindowList API 截图捕获 |
| `RegionSelectionWindow.swift` | 全屏覆盖层区域选择 |
| `AnnotationWindow.swift` | 标注窗口、工具栏、菜单栏 |
| `AnnotationView.swift` | 画布核心：绘制、交互、吸附、附着、Undo/Redo |
| `HitTestBuffer.swift` | 离屏 Hit Test 缓冲区（唯一颜色 Key 方案） |
| `Models.swift` | 所有数据模型和标注对象类 |

**核心设计：双图层 Color Picking**
- Layer A（可见层）：正常渲染所有标注对象
- Layer B（隐藏层）：每个对象用唯一颜色绘制，鼠标点击时读取像素颜色即可精确识别对象

## 构建与运行

```bash
# 构建 release 版本
swift build -c release

# 运行
.build/arm64-apple-macosx/release/AISnap
```

**要求：** macOS 13+，Swift 5.9+
